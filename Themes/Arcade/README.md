# Arcade Theme (BigMode)

This theme is inspired by an arcade cabinet layout and is intended for use with
Retromind's BigMode (controller-friendly UI).

## Artwork & Videos

This theme currently includes the following media files:

- `Images/cabinet.png`
- `Images/background.jpg`
- `Videos/bkg_anim.mp4`

These files were imported from or derived from assets shared in the
Attract-Mode community thread:

- http://forum.attractmode.org/index.php?topic=3979.0

Retromind does not claim authorship of these original media assets.
Theme integration and Avalonia layout work are provided by the Retromind project.

If you redistribute this theme, verify that your redistribution and usage of
the bundled media files comply with the original authors' terms.
