# Arcade Theme (BigMode)

This theme is inspired by an arcade cabinet layout and is intended for use with
Retromind's BigMode (controller-friendly UI).

## Artwork & Videos

The cabinet artwork and background video used in this theme are
based on assets from the Attract-Mode community:

- Forum thread: http://forum.attractmode.org/index.php?topic=3979.0

All credit for the original graphics and video goes to the respective authors
in the Attract-Mode community. The theme integration and Avalonia layout are
provided by the Retromind project.
