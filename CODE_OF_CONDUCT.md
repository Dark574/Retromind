# Code of Conduct

This project is intended to be a friendly and welcoming place for contributors and users.

By participating in this repository (issues, pull requests, discussions, etc.), you agree to follow this Code of Conduct.

## Our Pledge

We will act and interact in ways that contribute to an open, friendly, and respectful community.

## Expected Behavior

- Use welcoming and respectful language.
- Provide constructive feedback and be open to constructive feedback from others.
- Focus on what is best for the project and its users.
- Respect the time and effort of maintainers and contributors.

## Unacceptable Behavior

Examples of behavior that are not acceptable include:

- Personal attacks, insults, or demeaning comments.
- Threats or harassment.
- Spam, trolling, or intentionally disruptive behavior.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions,
- any other communication channels used by this project.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

If you experience or observe unacceptable behavior, you can:

- open an issue in this repository (if appropriate), or
- contact the maintainer privately (e.g. via the email address in the project profile).

Maintainers may take any action they consider appropriate, including:

- asking the user to change their behavior,
- closing or locking issues and pull requests,
- temporarily or permanently limiting a userâ€™s ability to participate in the project.
